# Neptune-HarmonyOS1.1-IOT

#### 介绍
Neptune开发板HarmonyOS V1.1源码仓
开源版本

#### 软件架构
本项目基于openharmony V1.1.0 LTS 版本，适配W800 WIFI/BT Combo 芯片。

#### 使用说明

1.  参考openharmony device 开发官网，安装V1.1 LTS 开发环境
2.  安装Neptune 开发工具链
